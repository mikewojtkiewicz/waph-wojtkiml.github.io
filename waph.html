This page introduces the WAPH course and the hands-on work completed as part of the front-end web development and security curriculum.

Course Overview

The course covers foundations of the web, front-end development, Ajax, API integration, secure PHP/MySQL development, and common web security attacks such as cross-site scripting, SQL injection, session hijacking, and CSRF.

Related Hands-on Projects

  • Lab 0: Development environment setup.
  • Lab 1: Foundations of HTTP, CGI, and basic web requests.
  • Lab 2: Front-end web development using HTML, CSS, JavaScript, jQuery, Ajax, and public APIs.
  • Hackathon 1: Cross-site scripting attack practice and analysis.
  • Individual Project 1: Professional profile website deployed through GitHub Pages.
